# SaanDaan

A Filipino-localized public transit companion app. Track buses in real-time, find nearby stops, and plan your commute.

## Features

- **Splash Screen** with SaanDaan branding
- **Onboarding** - 3-step introduction (Real-Time Awareness, Easy Navigation, Alerts & Convenience)
- **Login / Register** with social auth UI (Google, Facebook, Apple)
- **Home Screen** - Search, promo banner, services grid, categories, nearby stops
- **Live Tracking** - Track by Route or by Bus Number
- **Selected Bus Route** - Filter by time, view bus cards with status
- **Route Details** - Commuter line route with timeline stops
- **Profile** - Edit profile, settings, notifications/location toggles

## Getting Started

```bash
cd SaanDaan
npm install
npx expo start
```

Scan the QR code with **Expo Go** on your phone, or press `a` for Android / `i` for iOS simulator.

## Tech Stack

- React Native (Expo)
- React Navigation (Stack + Bottom Tabs)
- Ionicons (via @expo/vector-icons)
